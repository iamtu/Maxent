This dataset is strictly available only for academic research purposes.

The dataset is provided as tsv files, where fields 1,2 and 3 correspond to sentence id, sentence, and suggestion label respectively.
Label '0' stands for non-suggestion, and label '1' stands for suggestion, as per the annotation guidelines mentioned in the publication.
If you publish any results based on this dataset, please cite the publication: http://www.aclweb.org/anthology/D15-1258
